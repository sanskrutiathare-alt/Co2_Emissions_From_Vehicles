# Co2_Emissions_From_Vehicles
Project using Machine Learning Algorithms
This project uses Python to run two core machine learning models on a dataset. It uses Linear Regression to find relationships between variables and predict numerical values, and K-Nearest Neighbors (KNN) to group data into different categories. The script handles everything from splitting the data and scaling the numbers to training the models and printing out the final accuracy scores and a confusion matrix so you can see exactly how well both models performed.
